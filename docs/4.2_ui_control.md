# 4.2 User Interfaces Control
## Modules and Parameters
```lua
AddModule ("Function" [, "DisplayName"])
add modules to gui, returns the module pointer 

AddParameter (module, parent_node, "node", "description")
add parameter node to a module, returns node 

AddParameter (module, parent_node, "value", "description", default_value)
add a value parameter to a module, returns user input value 

AddParameter (module, parent_node, "range", "description", default_low, default_high)
add a range parameter to a module, returns user input low and high 

AddParameter (module, parent_node, "check", "description", true)
add a check parameter to a module, returns a boolean 

AddParameter (module, parent_node, "string", "description", "default")
add a string parameter to a module, returns a string 

AddParameter (module, parent_node, "choice", "description", "choice1|choice2|", default=1)
add a choice parameter to a module, returns a number 

AddParameter (module, parent_node, "file", "description", "C:\\default.txt")
add a file parameter to a module, returns a full file path 

AddParameter (module, parent_node, "scene", "description")
add a scene parameter to a module, returns a scene 

AddParameter (module, parent_node, "shapes", "description")
add a shapes layer parameter to a module, returns a shapes layer 

AddParameter (module, parent_node, "grid", "description")
add a grid parameter to a module, returns a grid 

AddParameter (module, parent_node, "table", "description")
add a table parameter to a module, returns a table 

AddParameter (module, parent_node, "field", "description")
add a table field parameter to a module, returns a string of the field name 

GetParameter (Shapes|Table|Grid|Module, "id")
returns the specified parameter 

SetParameter (Shapes|Table|Grid|Module, "id", number|"string"|pointer)
sets the specified parameter and returns ture if success 
```
## Data Basic Control
```lua
GetControl (".\\shapes.shp" [, ".\\grid.sgrd"] [, ".\\table.dbf"] [, ".\\scene.m3d" , ...])
opens shaps, table and grid data from disk and returns shapes,grid and table pointers by the order of inputs 

Open (".\\shapes.shp" [, ".\\grid.sgrd"] [, ".\\table.dbf"] [, ".\\scene.m3d" , ...])
opens shaps, table and grid data from disk and returns shapes,grid and table pointers by the order of inputs 

Close (Shapes|Grid|Table)
close data 

Update (Shapes|Grid|Table [, bShow=ture])
Update graphic interfaces and returns false if user cancel the current execution 

CreateView ("name", Shapes1|Grid1 [, Shapes2|Grid2, ...])
Create a window to show the objects arranged in layers 
```
## Other User Interfaces
```lua
Print ("Hello" [,...])
prints values to message window using tostring 

Note ("Hello" [,...])
notifies user and waits for acknowledgement 

SetProcessText ("Circle1")
SetProgress (Position, Range)

Sleep (Milliseconds)

GetReady ()
returns false if user cancel the current execution 

GetClickXY ()
returns x,y where user click last time 

GetDragExtent ()
returns leftbottom and righttop corner Lx, By, Rx, Ty that user drag last time 
```
